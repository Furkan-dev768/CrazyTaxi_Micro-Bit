Guide d'installation

BBC Micro:bit avec Arduino IDE

Suivez ce guide étape par étape pour configurer votre environnement de développement et programmer le micro:bit en langage C avec Arduino IDE.

Avant de commencer

Ce guide vous explique comment configurer votre environnement de développement pour programmer le micro:bit V2 en langage C. Assurez-vous d'avoir votre micro:bit et un câble USB à portée de main.

1

Télécharger Arduino IDE

Téléchargez et installez la dernière version de l'environnement de développement Arduino depuis le site officiel.

2

Ajouter le gestionnaire nRF5

Dans Arduino IDE :
Fichier → Préférences
Ajoutez cette URL dans "Gestionnaire de cartes supplémentaires" :

https://sandeepmistry.github.io/arduino-nRF5/package_nRF5_boards_index.json
3

Installer les cartes Nordic

Outils → Type de carte → Gestionnaire de cartes
Recherchez "nRF5" et installez :
Nordic Semiconductor nRF5 Boards par Sandeep Mistry

4

Sélectionner la carte

Outils → Type de carte → Nordic Semiconductor nRF5 Boards → BBC micro:bit V2

Assurez-vous de bien sélectionner la version V2.

5

Choisir le port

Connectez votre micro:bit via USB, puis :
Outils → Port
Sélectionnez le port correspondant (COM sur Windows, /dev/tty sur macOS/Linux)

6

Compiler et téléverser

Cliquez sur le bouton Téléverser (→) pour compiler et envoyer votre programme sur le micro:bit.

Le premier téléversement peut prendre quelques minutes.

Félicitations !

Votre environnement est configuré

Vous pouvez maintenant commencer à programmer votre micro:bit en langage C avec Arduino IDE.

Dépannage

Solutions aux problèmes courants

Le port n'apparaît pas

Essayez un autre câble USB. Certains câbles ne transfèrent que l'alimentation et pas les données.

Erreur de compilation

Vérifiez que vous avez bien sélectionné "BBC micro:bit V2" dans le menu Type de carte.

Échec du téléversement

Appuyez sur le bouton reset du micro:bit et réessayez le téléversement.
